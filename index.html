Projects

My First Game!

I called the first game I created "Cargo Run" because the objective of the game is to see how long you, the player, can last protecting a cargo ship in space. It does not use a game engine and was built using the pygame library. When making the game I took inspiration from space invaders and tried to modernize the classic game by creating a version that has updated graphics, better runtime, and powerups.

The code for this particular project was very inefficient and clunky. Looking back there are several things I would have done differently, but this project helped my grasp the full concept of object-oriented programming with classes which I now apply to all my programs. This game also made me realize the significance of the foundation of games provided by game engines. If I am going to build a game again I am interested in learning C# paired with the Unity Game Engine to create an RPG style experience.

COVID-19 Information App

All your COVID-19 data and news in one place

This app is available on android and was built as a school project. My goal when creating the app was to develop a platform to educated the public on COVID-19 facts utilizing easy to interperet information paired with an intuitive navigation layout. It provides data and news that can be toggled or customized between a local and global scale along with simplified explanations about testing and vaccine terminology.

Neural Network from Scratch

In the summer of 2020 I built my first neural network from scratch. It didn't accomplish any practical tasks, but my goal was to train the network to sort prime numbers without telling it explicitly how to. I have now learned that inorder to build networks to a modern standard, for example convolutional neural networks, it is much easier to re-train already existing models because it is to complex to solve practical problems from scratch. However, when making the structure of the network I gained in-depth knowledge about how weights and biases operate opposed to just inputs and outputs.

As I move forward, I want to begin learning about how AI can transform trading and asset transactions. My goal is to hopefully build a website that can provide analysis on ventures and risk assesment.
